## Comando para criar um projeto, React.Js (Typescript)

```js
npx create-react-app my-app --template typescript
```
ou

```js
npm install -g typescript 
```

Para verificar a version
tsc --version

para rodar o projeto
comando
npm start
ou
yarn start
